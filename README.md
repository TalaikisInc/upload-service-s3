# Upload Service S3

Work in progress.

## Install

```bash
npm i
amplify init
amplify add storage
```

## Start

```bash
npm run start
```

## Deploy

```bash
./slave_build.sh <name>
./slave_start.sh <name> <port>
```

## Test

```bash
npm run test
```
